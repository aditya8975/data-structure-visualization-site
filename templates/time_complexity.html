What's Time Complexity ?

To descide weather the given piece of code is good or not how would you tell. So the answer of this question is Time and space complexity .
Definition:   Time complexity is the computational complexity that describes the amount of computer time it takes to run algorithm. Similarly, Space complexity of an algorithm quantifies the amount of space or memory taken by an algorithm to run as a function of the length of the input.
Three Asymptotic
1. Big-θ (Big-Theta) notation

Let's look at a simple implementation of linear search: 

var doLinearSearch = function(array, targetValue) {
  for (var i = 0; i < array.length; i++) {
    if (array[i] === targetValue) { 
        return i;  // found 
    }
  }
  return -1;  // didn't find 
};

Let's denote the size of the array (array.length) by n nn. The maximum number of times that the for-loop can run is n nn, and this worst case occurs when the value being searched for is not present in the array.

Each time the for-loop iterates, it has to do several things:

  • compare i with array.length
  • compare array[i] with targetValue
  • possibly return the value of i
  • increment i.
